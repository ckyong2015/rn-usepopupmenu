
# React Native Popup Dialog for Android.


## Getting started

`$ npm install rn-usepopupmenu --save`


## Usage
```javascript
import UsePopupMenu from 'rn-usepopupmenu';

<UsePopupMenu 
    actions={['Edit', 'Remove']} 
    onPress={(e, index) => console.log(e, index)} 
/>
```

- icon
- color
- size
